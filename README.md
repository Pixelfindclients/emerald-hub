# EmeraldHub.dev

This is the central repository for the **Emerald Hub**, a unified dashboard for community tools and project resources.

## Deployment
This site is designed to be hosted via **Vercel** or **GitHub Pages**.

- **URL:** [https://emeraldhub.dev](https://emeraldhub.dev)
- **SSL:** Automatic via .dev requirements

## Project Structure
- `index.html`: Core structure and content.
- `style.css`: Emerald-themed visual styles.

## How to Update
1. Clone the repo: `git clone https://github.com/YOUR_USER/emerald-hub.git`
2. Make changes to `index.html`.
3. Push to main: `git push origin main`.
